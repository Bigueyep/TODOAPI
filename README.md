# TODO API

API REST de gestion de tâches développée avec FastAPI, PostgreSQL et SQLAlchemy.

## Features

- CRUD complet des tâches
- Gestion des priorités et statuts avec Enum
- Sous-tâches avec relation parent/enfant
- Filtres de recherche
- Validation des données avec Pydantic
- Docker & Docker Compose
- Tests avec Pytest
- Documentation Swagger/OpenAPI

---

# Stack technique

- FastAPI
- SQLAlchemy
- PostgreSQL
- Docker
- Pytest
- Pydantic
